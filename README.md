# Estoque Project

Sistema de controle de estoque desenvolvido em PHP e MySQL, seguindo arquitetura MVC, com autenticação de usuários, CRUD de produtos e boas práticas de segurança.

##  Tecnologias
- PHP 8
- MySQL
- HTML5, CSS3
- PDO (conexão segura)
- Estrutura MVC

##  Funcionalidades
- Login e controle de acesso
- Cadastro, edição e exclusão de produtos
- Validação de dados
- Hash de senhas e proteção contra SQL Injection
